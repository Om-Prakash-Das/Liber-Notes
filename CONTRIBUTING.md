# Contributing to Liber Notes

We welcome contributions to Liber Notes! Please read the guidelines below to make sure your contributions fit our project's needs:

## How to Contribute
1. **Fork the repository**: Click the fork button in the top right corner of this page.
2. **Clone your fork**: Clone your copy of the repository to your local machine using `git clone <your-fork-url>`.
3. **Create a branch**: Create a new branch for your feature or bug fix using `git checkout -b <branch-name>`.
4. **Make your changes**: Make your updates to the code/documentation.
5. **Commit your changes**: Use `git commit -m "Your commit message"` to save your changes.
6. **Push back to your fork**: Push your changes to your GitHub fork using `git push origin <branch-name>`.
7. **Submit a pull request**: Go to the original repository and click on the pull request button to submit your changes.

## Guidelines
- Make sure your code is well-documented.
- Follow the project's coding standards.
- Keep your changes as small as possible.

Thank you for your contributions!
